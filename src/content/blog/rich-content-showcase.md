Welcome to the rich content showcase post. This article is designed as a template to demonstrate how different formatting elements, media files, and layouts render on our blog detail page.

## 1. Typography & Headings

Use headings to structure your document. We support Heading 2, Heading 3, and Heading 4 for sections.

### Heading 3: Subsection

Paragraphs are formatted with a comfortable line-height and read color. Here is a demonstration of bold text, *italic text*, and `inline code block` elements.

#### Heading 4: Minor Point

This is a minor point under the subsection. Below is a block of unordered and ordered list items:

* **Minimalist Design:** Focus on negative space and clean typography.
* **Local First:** All user data stays inside the browser sandbox.
* **Performance:** Static generation via Astro guarantees fast loads.

Here is an ordered list for procedures:

1. Extract the raw text and images from the file.
2. Formulate the metadata and slug.
3. Output clean Markdown content.
